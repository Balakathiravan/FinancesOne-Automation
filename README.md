# FinancesOne Final Automation Framework

Tech stack:
- Java 17
- Maven
- Playwright Java (UI)
- TestNG
- RestAssured (API)
- ExtentReports (HTML report)

## How to run

1. Open this project as a Maven project in IntelliJ IDEA.
2. In terminal, run:

   mvn clean test

3. After execution, open `ExtentReport.html` in the project root to see the report.
